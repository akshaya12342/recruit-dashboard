Student Recruitment Dashboard (Django)
🔹 Overview

    This project is a simple web-based dashboard built using Django to manage student recruitment data.
    It allows users to add student details and view/search them based on course interest. 
    The application runs locally using SQLite as the database.

🔹 Features
    Add new students through a form
    View all students in a list format
    Search and filter students by course interest
    Django admin panel for managing data
🔹 Tech Stack
    Backend: Python, Django
    Database: SQLite
    Frontend: HTML, CSS, Bootstrap
